# **App Name**: Cafeteria

## Core Features:

- Landing Page: Display an appealing landing page with sections for the menu, about us, and contact information.
- Menu Display: Showcase the cafe's menu with item names, descriptions, prices, and appealing photographs.
- Reservation Form: Allow users to make reservations by selecting the date, time, and number of guests, and then submitting that request.
- Interactive Gallery: Offer an interactive gallery showcasing photos of the cafe's interior, food, and drinks.
- AI-Powered Ad Copy Generator: Generate promotional messages for the website using customer reviews as a tool to identify unique selling points.

## Style Guidelines:

- Primary color: Saturated mustard yellow (#E4B349) to reflect the warmth of a cafe setting.
- Background color: Desaturated light yellow (#F9F6EE) for a bright and inviting feel.
- Accent color: Light amber (#D98941) for interactive elements and calls to action.
- Headline font: 'Belleza' (sans-serif) for headings to convey style related to art and design, with 'Alegreya' (serif) for body text.
- Use elegant and custom-designed icons related to food, drinks, and cafe services.
- Employ a grid-based layout with generous white space and carefully chosen imagery.
- Subtle parallax scrolling effects and smooth transitions for a sophisticated browsing experience.
